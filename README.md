A Next.js + TypeScript web app that blends LLF’s bump vibes with on-demand AI-generated meme captions.

Pick a meme template (or search for a copyright-free image), choose a vibe like Late-Stage Capitalism or Alt Culture, select a desired tone, and let the bot write the chaos.

This project showcases front-end engineering, API design, AI integration, and creative UI/UX — all wrapped in my brand's aesthetic.
